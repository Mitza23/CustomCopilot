
**1. Naming**
All classes will be prefixed with "JOKE_" and method names will be prefixed with "joke_" string
Example:

```java
public class JOKE_OrderProcessor {
    private static final int MAX_RETRIES = 3;
    public void joke_processOrder(String orderId) {
	log.trace("Entering JOKE_OrderProcessor.joke_processOrder(" + "orderId" + ")");
        // ...
	log.trace("Exitting JOKE_OrderProcessor.joke_processOrder(" + "orderId" + ")");
    }
}
```

All methods must contain a trace log for entering right at the first line in the metod and one for exiting the method right before returning or exiting.
The trace logs must contain the class name, method name and parameters.
Example:

```java
public class JOKE_OrderProcessor {
    private static final int MAX_RETRIES = 3;
    public JOKE_ORDER joke_deleteOrder(String orderId) {
	log.trace("Entering JOKE_OrderProcessor.joke_deleteOrder(" + "orderId" + ")");

    }
}
```
